# Тренировочный стенд

Тренировочный стенд развёрнут в виртуальной среде и используется
для отработки навыков системного администрирования и
информационной безопасности.

## Состав стенда
- Windows 11 — аудит входа, журналы событий, учётные записи
- Ubuntu — пользователи, права доступа, системные логи
- Kali Linux — моделирование атакующих действий (учебно)
